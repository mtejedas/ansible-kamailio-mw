Ansible role for Kamailio (kamailio version >= 5.0)


## Using the role
### Installation
```
ansible-galaxy install ansible-kamailio-mw
```

### Example Playbook
```
  - hosts: all
    roles:
      - ansible-kamailio-mw
```

### Variables

See [`defaults/main.yml`](defaults/main.yml).
